Python Fundamentals

Variables and Data Types

Python is dynamically typed and supports various basic data types.

# Basic data types
name = "Alice" # str
age = 25 # int
height = 1.75 # float
is_student = True # bool
has_pet = None # NoneType

# Type checking
print(type(name)) #
print(type(age)) #

Control Flow

Python uses indentation to define code blocks in control structures.

# If statement
if age >= 18:
    print("Adult")
elif age >= 13:
    print("Teenager")
else:
    print("Child")

# For loop
for i in range(5):
    print(i)

Data Structures

Lists and Tuples

Python provides several built-in data structures for storing collections.

# Lists (mutable)
fruits = ['apple', 'banana', 'orange']
fruits.append('grape')
fruits[0] = 'pear'

# Tuples (immutable)
coordinates = (10, 20)
# coordinates[0] = 30 # This would raise an error

Dictionaries and Sets

Key-value pairs and unique collections in Python.

# Dictionary
person = {
    'name': 'Bob',
    'age': 30,
    'city': 'New York'
}

# Set
unique_numbers = {1, 2, 3, 3, 4} # {1, 2, 3, 4}

Functions

Function Definitions

Functions in Python are defined using the def keyword.

# Basic function
def greet(name):
    return f"Hello, {name}!"

# Function with default parameter
def power(base, exponent=2):
    return base ** exponent

# Lambda function
square = lambda x: x ** 2

Object-Oriented Programming

Classes and Objects

Python supports class-based object-oriented programming.

class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        return f"{self.name} says Woof!"

# Creating an object
my_dog = Dog("Rex")
print(my_dog.bark()) # Rex says Woof!

Modules and Packages

Working with Modules

Python's module system helps organize and reuse code.

# Importing modules
import math
from datetime import datetime

# Using module functions
print(math.pi) # 3.141592653589793
print(datetime.now()) # Current date and time
